# Frix - Portal de Cliente

Esta es una aplicación de Next.js para Frix, un servicio de desarrollo web. Funciona como un portal para que los clientes puedan ver sus proyectos e interactuar con el servicio.

## Stack de Tecnologías

Este portal de cliente está construido con las siguientes tecnologías:

- **Framework:** [Next.js](https://nextjs.org/)
- **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
- **Componentes UI:** [ShadCN UI](https://ui.shadcn.com/)
- **Alojamiento:** Aunque está configurado para [Firebase App Hosting](https://firebase.google.com/docs/app-hosting), este proyecto puede ser desplegado fácilmente en plataformas como [Vercel](https://vercel.com/).

Para empezar, echa un vistazo a `src/app/page.tsx`.
