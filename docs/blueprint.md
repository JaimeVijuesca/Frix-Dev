# **App Name**: Frix

## Core Features:

- Vitrina de Servicios: Página de inicio que muestra los servicios web ofrecidos, ejemplos y el proceso de trabajo, destacando la rapidez y sencillez en la creación de webs con la tecnología.
- Autenticación Segura: Sistema de autenticación de usuarios que admite el inicio de sesión por correo electrónico y Google a través de Firebase.
- Panel de Administración: Panel de administración para que el desarrollador gestione clientes, proyectos y contenido.
- Portal del Cliente: Panel específico para que el cliente vea el estado y las actualizaciones de su proyecto web.
- Gestión de Proyectos: Seguimiento de proyectos a través de estados: pendiente, en progreso y entregado.
- Subida de Recursos: Permitir la carga de recursos como logotipos, imágenes y documentos utilizando Firebase Storage.
- Formulario de Contacto: Formulario de contacto y envío de solicitudes web.

## Style Guidelines:

- Color primario: Deep Indigo (#3F51B5) para transmitir profesionalismo y confiabilidad.
- Color de fondo: Gris claro (#F5F5F5) para una sensación limpia y moderna.
- Color de acento: Teal (#009688) para resaltar acciones clave y elementos interactivos.
- Fuente del cuerpo y del titular: 'PT Sans' (sans-serif) para facilitar la lectura y dar un toque moderno.
- Fuente de código: 'Source Code Pro' para mostrar fragmentos de código.
- Utilice un conjunto de iconos sencillos y profesionales de una biblioteca como Material Icons para representar los servicios y las acciones.
- Emplee un diseño limpio basado en cuadrículas con mucho espacio en blanco para garantizar un aspecto profesional.
